# Resilient Connect

A lightweight experimental communication system designed for resilient peer-to-peer networking during unstable or restricted internet access.

## Features

- Peer-to-peer messaging
- Relay fallback support
- Local-first architecture
- Simple encryption layer
- Offline message storage
- Minimal dependencies

## Stack

- Python
- FastAPI
- WebSockets
- SQLite

## Run

```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
./scripts/run.sh
```
